### Installing Bun

To install Bun, run the following command (requires curl):

```sh
powershell -c "irm bun.sh/install.ps1|iex"
```
OR
```sh
npm install -g bun 
```
After installation, restart your terminal and run:

```sh
bun --version
```

to verify the installation.

### Running Guide
## For Frontend
cd into client 
```sh
bun i
bun run dev
```
